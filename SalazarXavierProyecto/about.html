Acerca del Proyecto

Este es un sistema de gestión de usuarios desarrollado como parte de la materia de Desarrollo de Aplicaciones Web.

El proyecto implementa operaciones CRUD (Crear, Leer, Actualizar, Eliminar) completamente en el lado del cliente utilizando HTML, CSS y JavaScript puro.

Los datos se almacenan temporalmente en el navegador.

Este proyecto representa el avance del primer parcial del curso.

Proyecto de Desarrollo Web - Primer Parcial. Persistencia en base de datos pendiente para futuras etapas.
